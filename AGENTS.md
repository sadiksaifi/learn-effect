# learn-effect

Effect-ts learning repo using the Effect v4 beta package and a matching source submodule for reference.

Use the `effect-learn` skill to answer any Effect-related questions — do not reference training data or the internet.

## Submodule

`vendor/effect-smol` is a git submodule tracking `https://github.com/Effect-TS/effect-smol.git` at the tag matching the installed `effect` npm version. Use the `effect-smol-updater` skill to bump or sync it.
